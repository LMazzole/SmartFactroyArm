bool MyServo::moveToPosition ( int  desiredposDeg)

Move servo to desired position with 1 degree per functioncall.

Max. Resolution from read() is only about 15 degree. This will cause an slight Position-Error in the first functioncall. The Servo will get detached when the position is reached.

Parameters
pos- Desired Servo-position in degree; -90deg is left 0deg is Center, 90deg is right
Returns
true - if servo is in position
false - if servo is not in position
